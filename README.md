# Equalijobs

Equalijobs é uma plataforma que visa empregabilidade de pessoas com deficiência.

## Instalação

Use o gerenciador de pacotes [npm](https://docs.npmjs.com/cli/v6/using-npm) para instalar o Equalijobs.

```
npm install && npm start
```
