# Equalijobs

Equalijobs é uma paltaforma que visa empregabilidade de pessoas com deficiência.

## Installation

Use the package manager [npm](https://docs.npmjs.com/cli/v6/using-npm) to install foobar.

```bash
npm install && npm start
```
